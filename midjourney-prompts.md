## Tips for Best Results:
- Use `--niji 6` for best anime/cute style
- Use `--ar 3:4` for portrait orientation (perfect for cards)
- Use `--style cute` to emphasize cuteness
- Add "pastel background" to make them pop
- Include "chibi proportions" for that kawaii look
- Add "sparkles" or "hearts" for extra kawaii vibes

## Generic Formula:
```

```

## After Generating:
1. Download your favorites

3. Add them to your repo in an `/images/` folder
4. Update the HTML to use local paths: `images/space-marine-cute.png`
